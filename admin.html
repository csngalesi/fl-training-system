Guia de Animação

📐 Geometria do Campo

  • O campo simula 20m × 10m em escala real.
  • Eixo X: 0 (esquerda) → 10 (direita)
  • Eixo Y: 0 (gol / baixo) → 20 (fundo / cima)
  • Gol fixo centralizado em {"x": 5, "y": 0}

🧑 Atores (IDs)

p1 — Vermelho p2 — Azul p3 — Verde p4 — Âmbar ball — Bola

📋 Passo de Movimento

Cada passo move um ator para uma coordenada em um tempo (dur) em milissegundos.

{"actor": "p1", "to": {"x": 3, "y": 10}, "dur": 1200}

🔀 SYNC — Mover dois ao mesmo tempo

Por padrão o motor espera o ator terminar antes do próximo passo. Use sync: true no primeiro passo para que ambos se movam juntos:

{"actor": "ball", "to": {"x": 3, "y": 10}, "dur": 800, "sync": true}, {"actor": "p1", "to": {"x": 4, "y": 12}, "dur": 1200}
Sem sync: true, o passe ficará travado esperando o jogador terminar de correr.

⏱ DELAY — Pausa geral

Pausa tudo por N milissegundos. Ideal para o "pisão" do pivô ou tomada de decisão.

{"delay": 300}

💡 Exemplo completo

[ {"actor": "ball", "to": {"x": 5, "y": 15}, "dur": 1000}, {"delay": 400}, {"actor": "ball", "to": {"x": 2, "y": 10}, "dur": 800, "sync": true}, {"actor": "p1", "to": {"x": 2, "y": 10}, "dur": 1200}, {"delay": 200}, {"actor": "ball", "to": {"x": 5, "y": 0}, "dur": 800} ]

🏟 Setup — Posições Iniciais

{ "players": { "p1": {"x": 5, "y": 15}, "p2": {"x": 2, "y": 10}, "p3": {"x": 8, "y": 10}, "p4": {"x": 5, "y": 5} }, "ball": {"x": 0.5, "y": 0}, "cones": [{"x": 4, "y": 8}, {"x": 6, "y": 8}] }
